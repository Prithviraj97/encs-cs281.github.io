Course Description

This course builds upon the fundamentals learned in CS180, introducing students to more advanced programming concepts including object-oriented design, data structures, algorithm analysis, and software engineering principles.

By the end of this course, students will be able to design, implement, and analyze algorithms and data structures to solve computational problems.
